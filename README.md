| 1 | 初始化工程 | android studio<br>git |

## 教学建议

1. 课堂老师至少要完成三件事：上次课复习、本次课讲解及实验、课堂实践
2. 课后学生至少要做两件事：完成相应的课后实践任务，总结所学知识点
2. 学期结束前，**每位同学**必须在现有的天气预报基础上，至少增加或完善3个功能点。并提交一个属于自己的天气预报APP。
